# Mikrotik CHR + Ansible RADIUS Configuration

## Overview
This project demonstrates the deployment of Mikrotik CHR using VirtualBox, firewall configuration via Winbox, RADIUS configuration using Ansible, and Hotspot setup.

## Steps Performed
1. Deployed Mikrotik CHR on VirtualBox
2. Configured firewall rule blocking TCP 443 to 1.1.1.1
3. Installed Ansible and configured RADIUS server
4. Verified RADIUS via Winbox
5. Added second interface and configured Hotspot
6. Exported configuration and documented work

## Challenges
- Enabling API access for Ansible
- Network interface mapping in VirtualBox
